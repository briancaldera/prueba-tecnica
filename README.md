# Prueba Técnica

## 📦 Instalación

```bash
git clone https://github.com/briancaldera/prueba-tecnica.git
```
## ⚙️ Cómo ejecutar el proyecto

### Inicia los servicios

```bash
bash prueba up
```

### Migra la base de datos

```bash
bash prueba migrate
```

### Ejecuta las pruebas

```bash
bash prueba run
```

### Finaliza los servicios

```bash
bash prueba down
```
